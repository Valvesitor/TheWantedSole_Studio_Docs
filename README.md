# The Wanted Sole Studio Docs

Bem-vindo à documentação oficial da **The Wanted Sole Studio**.

Esta documentação foi organizada para funcionar no **GitBook**, com cada script em sua própria área e com subpáginas separadas para instalação, licença, uso, configuração e suporte.

No GitBook, as subpáginas ficam aninhadas dentro da categoria principal na sidebar. Assim, a documentação fica limpa e cada produto tem sua própria área.
{% endhint %}
