# Support

Antes de abrir ticket, peça para o comprador enviar:

- Prints ou vídeos do problema.
- Nome exato do resource.
- Erro do console.
- Erro do F8, se existir.
- Versão do pack.
- Framework usado no servidor.

{% hint style="info" %}
Quanto mais detalhes o comprador enviar no primeiro contato, mais rápido o suporte consegue identificar o problema.
{% endhint %}
