# Installation

Depois de baixar o pack, coloque a pasta do resource dentro de `resources/` e adicione no `server.cfg`:

```cfg
ensure TWS_Custom_Ped_Pack
```

{% hint style="warning" %}
Altere o nome do resource acima para o nome real da pasta enviada ao comprador.
{% endhint %}

## Passo a passo

1. Extraia o arquivo recebido.
2. Copie a pasta para `resources/`.
3. Confirme se o nome da pasta não contém espaços ou caracteres especiais.
4. Adicione o `ensure` no `server.cfg`.
5. Reinicie o servidor.
