# TWS Identity Forge

Guia completo de instalação, configuração e uso do editor de outfit para RedM.

- **Status:** Ativo
- **Versão:** v2.0.7
- **Plataforma:** RedM

## Visão geral

O **TWS Identity Forge** centraliza criação, pré-visualização, validação e exportação de outfits para servidores RedM.

## Recursos principais

- Monta looks completos a partir de **drawables e variações de cor**.
- Exibe um **ped de pré-visualização** ao lado do personagem ou em local configurado.
- Importa, edita e **exporta XML de outfit** com validador avançado item a item.
- Guarda **projetos e slots de outfit** no servidor com persistência por jogador.
- Suporta **presets CS** em arquivos XML no padrão `cs_*.ymt.xml`.
- Interface em **pt-BR, en-US e es-ES** com idioma salvo entre sessões.
- Busca inteligente com filtro por favoritos: `fav:`, `@fav` ou `★`.

## Requisitos

| Requisito | Detalhe |
| --- | --- |
| Jogo | RedM / Red Dead Redemption 2 Roleplay. |
| fx_version | `cerulean`. |
| MetaPed Resource | Resource que aplica outfits no servidor, como VORP, RSG ou equivalente. |
| Licença | Servidor com saída HTTPS e convars configuradas quando a validação remota estiver ativa. |
