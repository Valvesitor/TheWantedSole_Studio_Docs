# Usage

Esta página explica os comandos e principais funções da interface.

## Comandos

| Comando | Função | Uso |
| --- | --- | --- |
| `/creator` | Abre ou fecha a interface principal do Studio. | Jogador |
| `/roupas` | Abre o menu simples de roupas por componente, estilo e textura. | Jogador |
| `/repositionpreview` | Recoloca o ped de pré-visualização. Aceita distância e altura opcionais. | Jogador |
| `/tws_cs_test` | Solicita a lista de presets CS ao servidor para debug. | Debug |
| `Tws_rescan_peds` | Força novo scan de peds customizados listados pelo servidor. | Console |
| `tws_info <id>` | Mostra quantidade de projetos/outfits salvos no KVP do jogador. | Console |

## Interface

### Preview

Escolha o modelo do ped, use modo estático e ajuste a iluminação de pré-visualização.

### Favoritos

Marque itens com estrela e filtre usando `fav:`, `@fav` ou `★`.

### XML

Importe, cole, valide e exporte XML de outfit com mensagens de erro por item/tag.

### Projetos

Salve conjuntos de trabalho, use slots de outfit e carregue presets CS.
